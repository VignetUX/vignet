# Vignet

Vignet (pronounced like "vignette") is a frontend scenes workshop generated directly from your Vitest tests.

This allows you to create frontend scenes of your web app states for full pages as well as design system components using the Vitest features you're already familiar with.

## Overview and Getting Started

See the main docs page at [vignet.dev](https://www.vignet.dev).

## Running the example

An example react frontend exists in `example/frontend`. To run this example follow these steps:

1. Build Vignet via `pnpm build`
2. `cd example/frontend` then `pnpm vignet`. Your browser will then open a Vignet window.
